# Конспект курса ChatGPT Prompt Engineering for Developers

Курс Andrew Ng и Isa Fulford.

В репозитории содержатся структурированные заметки по курсу *ChatGPT
Prompt Engineering for Developers*.

## Основные темы

-   принципы prompt engineering
-   итеративная разработка промптов
-   суммаризация
-   извлечение информации
-   преобразование текста
-   генерация текста
-   создание чат‑ботов

## Основные принципы

1.  писать четкие инструкции
2.  давать модели время на размышление

## Техники промптинга

### Разделители

Используйте специальные символы для отделения инструкций от текста.

### Структурированный вывод

Запрашивайте ответы в формате JSON, HTML или таблиц.

### Few‑shot промптинг

Предоставляйте примеры желаемого результата.

### Пошаговое рассуждение

Поощряйте модель решать задачу шаг за шагом.

### Минимизация галлюцинаций

Просите модель сначала извлекать цитаты из текста.

## Итеративная разработка

Цикл разработки:

1.  идея
2.  тестирование
3.  анализ
4.  улучшение

## Параметр Temperature

Temperature управляет креативностью.

-   Temperature = 0 → детерминированный результат
-   Temperature ≥ 0.7 → больше креативности

## Пример чат‑бота

``` json
  "drinks": [],

}
```
